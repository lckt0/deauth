COMMANDS OVERVIEW

Setup Config Modules General

create

Creates a new verification panel and enables verification in server.

remove

Removes all verification components in server.

edit

Edits the current config property.

reset

Resets the config to default.

export

Creates a template key for config. Allows you share or backup the config to anywhere.

import

Imports a config from another server.

agelimit

Configures a account age limit for banning young accounts.

cdis

Blocks joins from specific countries to server.

antiraid

Turns on/off anti raid module.

ban

Bans the member.

unban

UnBans a user.

lock

Locks your server to outside.

whois

View the basic informations & verificiton status about user.

help

Display help.

logs

See last verification attempts.

cleanup

Built-in preset actions to batch managing of members.
